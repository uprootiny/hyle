No vendor lock-in

Switch models anytime. Export sessions. Code is open source.

Sessions are stored as JSON in ~/.local/state/hyle/. You can read, modify, or delete them. Switch between Claude, GPT-4, Llama, or Mistral with a single command. The source code is MIT licensed.
.

Your local environment

Runs in your terminal. Reads your files. Uses your tools.

hyle runs as a single binary in your terminal. It accesses only the files you work with. Bash commands run with your permissions. No cloud sync, no project upload, no dependency on external services beyond the LLM API.
