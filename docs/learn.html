Learn by building

An AI coding assistant that explains what it's doing. Perfect for learning new languages, frameworks, and patterns.

1

Install in one command

Just Rust and a free API key. No complex setup.

$ cargo install --git https://github.com/uprootiny/hyle
$ hyle config set key YOUR_FREE_KEY
2

Describe what you want to build

Use natural language. hyle understands context.

> make a simple todo app in rust
> explain each step as you go
3

Watch and learn

hyle reads files, writes code, runs tests. Every action is visible.

# hyle shows each tool call:
read(path="src/main.rs")
write(path="src/todo.rs", ...)
bash(command="cargo test")
4

Ask questions anytime

Stuck? Ask why. The conversation continues.

> why did you use a Vec instead of an array?
> show me another way to do the error handling
