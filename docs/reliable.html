Code assistance with built-in safeguards

Rust-native autonomous assistant with atomic writes, automatic backups, and guardrails against destructive operations. Designed for production environments.

View Documentation GitHub

Capabilities

Autonomous Execution

Reads, writes, and patches files. Executes shell commands. Iterates until task completion with configurable iteration limits.

Session Persistence

Conversations persist across restarts. Context preserved. Integrates with existing workflows via JSON export.

Model Flexibility

35+ models via OpenRouter. Rate limit detection with automatic fallback. Health tracking per model.

Telemetry

Token usage tracking. Latency traces. Context budget visibility. Cost estimation for paid models.

Safety Features

Destructive Command Blocking

rm -rf and similar patterns blocked by default

Atomic File Writes

Write to temp, fsync, rename. No partial writes.

Write Verification

Read-back check after every file write

Backup Rotation

Timestamped backups, keeps last 3 versions

Rate Limit Handling

Auto-switches to fallback models on 429

Session Auto-save

Work preserved on Ctrl-C or crash
