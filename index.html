How it works

Your sketch is submitted to a GitHub Actions workflow. hyle reads it, scaffolds the project, writes code, runs tests, and deploys to a subdomain. You get a link when it's done.

Powered by free LLMs via OpenRouter. No API key needed. Source code.

Run locally

cargo install --git https://github.com/uprootiny/hyle
hyle "add error handling to src/main.rs"
