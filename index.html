Installation

From Source (Recommended)

$ git clone https://github.com/uprootiny/hyle.git
$ cd hyle
$ cargo build --release
$ cp target/release/hyle ~/.local/bin/

Configuration

$ export OPENROUTER_API_KEY="sk-or-..."

# Or add to your shell config
$ echo 'export OPENROUTER_API_KEY="sk-or-..."' >> ~/.bashrc

Usage Examples

Interactive TUI Mode

$ hyle

One-shot Query

$ hyle "explain this codebase"

Pipe Input

$ cat src/main.rs | hyle "review this code"

Specific Model

$ hyle --model mistralai/devstral-small "fix the bug"

List Available Models

$ hyle models
