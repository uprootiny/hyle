hyle

code that writes itself

Rust-native autonomous code assistant. Paste a project sketch, get a working codebase. No boilerplate, no busywork.

# install
cargo install hyle

# run
hyle "build me an API server with auth"

# or paste a multi-thousand-line sketch
hyle orchestrate

Autonomous

Agent loop with tool execution. Builds, tests, iterates until done.

Minimal

Unix philosophy. Single binary. No daemon. Text in, code out.

Fast

Rust-native. Sub-second startup. Multi-model fallback.

Open

MIT licensed. OpenRouter-powered. Bring your own key.
