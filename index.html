Installation

Up and running in minutes

Choose your preferred installation method.

1

Install Rust (if needed)

$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh $ rustup default stable
2

Build from source

$ git clone https://github.com/uprootiny/hyle.git $ cd hyle $ cargo build --release $ mkdir -p ~/.local/bin $ cp target/release/hyle ~/.local/bin/
3

Configure your API key

# Add to your shell profile (~/.bashrc, ~/.zshrc, etc.) $ export OPENROUTER_API_KEY="sk-or-v1-..."

Usage

Simple by design

Just type what you want to do.

Interactive TUI mode
$ hyle
One-shot query
$ hyle "explain this codebase"
Pipe input
$ cat src/main.rs | hyle "review this"
Specific model
$ hyle --model mistralai/devstral-small
List models
$ hyle models
Doctor check
$ hyle doctor
