hyle

autonomous code assistant

Paste a project sketch. Get working code. Rust binary, no runtime, runs anywhere.

Install

cargo install --git https://github.com/anthropics/hyle
hyle config set key YOUR_OPENROUTER_KEY

Requires Rust. Get a free API key at openrouter.ai.

Usage

# Interactive session
hyle

# One-shot task
hyle "add error handling to src/main.rs"

# Project orchestrator (paste large sketches)
hyle orchestrate

What it does

Agentic loop
Reads files, writes code, runs tests, iterates until done.
Session persistence
Resume conversations. Context survives restarts.
Slash commands
/build, /test, /diff, /commit — 30+ local commands.
Multi-model fallback
Rate limited? Automatically switches to free models.
Project orchestration
Paste a sketch, get scaffolding + deployment configs + autonomous build.

What it doesn't do

  • No telemetry. No cloud dependency. Your code stays local.
  • No daemon. Starts instantly, exits cleanly.
  • No framework lock-in. Works on any codebase.
